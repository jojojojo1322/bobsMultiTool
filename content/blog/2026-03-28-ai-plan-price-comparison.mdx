## 한눈에 비교

기준일은 2026-07-03이다. AI 플랜은 이름보다 결제 위치와 사용 흐름을 먼저 나눠야 한다. 같은 `$20`이라도 브라우저 챗, 긴 문서, Google 문서·메일, IDE 코딩에서 체감이 다르다.

## 기준일과 확인한 곳

AI 가격표는 몇 달 단위로도 바뀐다. 나라별 가격, 부가세, 앱스토어 결제, 팀 계정, 연간 할인, 프로모션, 사용량 크레딧 전환에 따라 실제 결제 금액이 달라진다. 그래서 이 글은 "영원한 정답"이 아니라 2026-07-03에 결제창을 열기 전에 보는 확인 메모다.

확인한 곳은 [ChatGPT pricing](https://chatgpt.com/pricing/), [OpenAI ChatGPT Plus 도움말](https://help.openai.com/en/articles/6950777-what-is-chatgpt-plus), [OpenAI ChatGPT Pro 등급 도움말](https://help.openai.com/en/articles/9793128-about-chatgpt-pro-tiers), [Claude 플랜 안내](https://support.claude.com/en/articles/11049762-choose-a-claude-plan), [Google AI plans](https://one.google.com/intl/en_us/about/google-ai-plans/), [Google AI 구독 페이지](https://gemini.google/subscriptions/), [GitHub Copilot 가격표](https://github.com/features/copilot/plans), [GitHub Copilot usage-based billing 안내](https://github.blog/news-insights/company-news/github-copilot-is-moving-to-usage-based-billing/)다. 공개 비교 글과 커뮤니티 글은 사람들이 헷갈리는 지점, 예를 들면 ChatGPT Pro 100달러/200달러 이름, Copilot 크레딧 전환, Claude Max 비용 체감 같은 질문을 파악하는 데만 썼고, 금액은 공식 페이지를 우선했다.

## 먼저 나눌 질문

| 질문 | 예라고 답하면 | 먼저 볼 플랜 |
| --- | --- | --- |
| 브라우저에서 글, 파일, 이미지, 리서치를 섞어 쓰는가 | 한 계정에서 여러 작업을 처리해야 한다 | ChatGPT Plus 또는 Claude Pro |
| 긴 문서와 차분한 글 다듬기가 많은가 | 답변을 읽고 고치는 시간이 길다 | Claude Pro, 이후 Max |
| Gmail, Docs, Drive, NotebookLM 안에서 바로 쓰는가 | 저장공간과 문서 생태계가 같이 필요하다 | Google AI Pro |
| 코딩이 IDE 안에서 끝나는가 | 자동완성, code review, cloud agent가 중요하다 | GitHub Copilot Pro |
| 사용량 제한 때문에 하루에 몇 번씩 멈추는가 | 이미 무료/20달러 플랜을 충분히 썼다 | Pro, Max, Ultra, Copilot Max 후보 |
| 아직 습관이 없다 | 이번 주에 몇 번 열지 모른다 | 무료 또는 한 달짜리 20달러 이하 |

이 질문을 먼저 두는 이유는 가격이 비슷해도 제품 성격이 다르기 때문이다. ChatGPT Plus와 Claude Pro는 둘 다 `$20`대지만, 한쪽은 파일·이미지·리서치·Codex까지 한 계정에서 넓게 쓰는 느낌이고, 다른 한쪽은 긴 문서와 대화 흐름에서 장점이 보일 때가 있다. Google AI Pro는 단순 챗보다 Google One 저장공간과 Gmail, Docs, NotebookLM, Flow를 같이 쓰느냐가 핵심이다. Copilot은 브라우저 챗 가격표로 비교하면 헷갈리고, IDE와 GitHub 안에서 실제로 얼마나 자주 코드를 고치는지로 봐야 한다.

## ChatGPT 쪽은 Plus와 Pro 사용량을 분리한다

ChatGPT 공식 가격표는 Free, Go, Plus, Pro, Business, Enterprise로 나뉜다. 개인 사용자가 먼저 보는 줄기는 Free, Go, Plus, Pro다. 공식 가격표와 도움말 기준으로 Plus는 `$20/month`이고, Pro는 `$100`와 `$200` 두 등급이 있다. OpenAI 도움말은 두 Pro가 핵심 기능은 같고 사용량 허용량이 다르다고 설명한다. `$100` Pro는 Plus보다 5배, `$200` Pro는 Plus보다 20배 높은 사용량 쪽이다.

그래서 ChatGPT는 "무조건 Pro가 좋다"가 아니라 "Plus에서 무엇이 막혔나"를 먼저 본다. 파일 업로드가 부족했는지, Codex 작업이 부족했는지, deep research가 부족했는지, 긴 문맥이 부족했는지, 아니면 그냥 한 달에 며칠만 많이 쓴 것인지가 다르다. 한 달에 며칠만 몰아서 쓰는 사람은 `$100` 이상으로 올리기 전에 Plus에서 작업 날짜와 막힌 지점을 기록하는 편이 낫다.

Go도 따로 본다. Go는 Free보다 더 긴 대화와 더 많은 업로드·이미지 생성 쪽으로 보이지만, 공식 가격표에 광고 포함 가능성이 표시된다. 매일 AI를 켜는 습관이 없으면 Go도 급하지 않다. 반대로 무료에서 메시지·업로드가 자주 막히지만 Plus의 고급 기능까지는 필요 없으면 Go가 징검다리일 수 있다.

## Claude는 Pro에서 막혔을 때 Max를 본다

Claude 공식 플랜 안내는 Free, Pro, Max 5x, Max 20x를 개인용 핵심 줄기로 보여준다. Pro는 `$20/month` 또는 `$200/year`이고, Max 5x는 `$100/month`, Max 20x는 `$200/month`다. 표기상 Max 5x는 Pro보다 5배, Max 20x는 20배 용량을 보는 구조다.

Claude는 긴 문서, 글 다듬기, 코드 설명, 차분한 검토에서 손에 맞는 사람이 있다. 하지만 Max부터 고르는 건 위험하다. 먼저 Pro에서 실제로 막혔는지 봐야 한다. 예를 들어 하루에 긴 문서를 여러 번 넣다가 세션 용량이 자주 막힌다면 Max를 볼 이유가 있다. 반대로 일주일에 몇 번 문장 다듬는 정도라면 Pro도 매달 고정비가 된다.

공개 기사와 사용자 글에서 반복되는 질문은 "Max 20x가 정말 `$200` 값을 하느냐"다. 답은 사람마다 다르지만 기준은 단순하다. Claude가 산출물이나 작업시간을 실제로 줄여주고, Pro 제한 때문에 그 흐름이 자주 끊기면 Max가 후보가 된다. 제한을 만난 기억이 없으면 아직 Max의 문제가 아니다.

## Google AI는 저장공간과 앱 안 기능을 같이 본다

Google AI plans는 Gemini만 따로 사는 느낌보다 Google One 저장공간과 앱 안 기능을 같이 보는 쪽에 가깝다. 공식 페이지 기준으로 AI Plus는 400GB 저장공간, AI Pro는 5TB 저장공간, Ultra는 20TB 또는 30TB 저장공간 축으로 나뉘며, Gemini, Gmail, Docs, NotebookLM, Flow, AI Studio, Google Antigravity 같은 기능 제한이 같이 달라진다. 가격은 지역별 페이지와 결제창에서 다르게 보일 수 있으므로 마지막에는 내 계정의 Google One 화면을 확인해야 한다.

Google AI Pro가 맞는 사람은 "AI 챗을 하나 더 쓰고 싶다"보다 "Gmail과 Docs에서 바로 쓰고, Drive 저장공간도 필요하다"에 가깝다. 이미 Google One 저장공간을 돈 내고 쓰고 있다면 추가 비용 계산이 달라진다. 반대로 저장공간이 필요 없고 브라우저 챗만 쓰는 사람은 `$19.99` 근처라도 체감이 약할 수 있다.

Ultra는 더 조심해야 한다. Google 공식 페이지는 Ultra를 더 높은 Gemini 사용량, Flow, AI Studio, Jules, Google Antigravity, YouTube Premium, 큰 저장공간 같은 묶음으로 설명한다. 영상 생성, 개발 실험, Google 생태계 안 agent 기능을 매주 쓰는 사람이 아니면 가격표만 보고 바로 고를 단계는 아니다.

## Copilot은 가격보다 크레딧과 IDE 흐름을 본다

GitHub Copilot 공식 가격표는 Free, Pro, Pro+, Max를 보여준다. 개인 플랜 기준으로 Pro는 `$10/month`, Pro+는 `$39/month`, Max는 `$100/month`다. 가격표에는 Pro의 cloud agent와 code review, code completion, 모델 선택, 3rd party agents, 월 크레딧이 같이 표시된다. Pro+와 Max는 premium model, 더 높은 포함 사용량, priority access, high-volume agent workflow 쪽으로 올라간다.

여기서 중요한 건 Copilot이 사용량 기반 과금으로 움직이고 있다는 점이다. GitHub의 usage-based billing 안내는 Pro와 Pro+가 월 구독료에 맞춘 AI Credits로 전환된다고 설명한다. 그런데 가격표에는 Pro, Pro+, Max의 총 크레딧이 별도로 보인다. 이 차이는 전환 시점, 계정 상태, 프로모션, 플랜 가능 여부에 따라 헷갈릴 수 있다. 그래서 Copilot은 블로그 글만 보고 결제하지 말고, 실제 GitHub billing 화면에서 내 계정의 포함 크레딧과 추가 사용 한도를 확인해야 한다.

Copilot이 맞는 사람은 "AI와 대화하고 싶다"보다 "파일을 열어 고치고, PR을 보고, IDE 안에서 바로 바꾸고 싶다"에 가깝다. 브라우저 챗으로 글과 자료를 다루는 시간이 더 길면 ChatGPT나 Claude가 먼저일 수 있다. 반대로 하루 대부분을 VS Code, JetBrains, Xcode, Neovim, GitHub PR에서 보낸다면 Copilot Pro부터 테스트하는 게 현실적이다.

## 한 달 실험표

| 이번 달 작업 | 먼저 결제할 후보 | 실패 기준 |
| --- | --- | --- |
| 블로그·문서·리서치·파일 분석을 매일 한다 | ChatGPT Plus | 일주일에 3일 이상 열지 않으면 해지 |
| 긴 원고·기획서·코드 설명을 많이 읽힌다 | Claude Pro | 무료와 체감 차이가 적으면 해지 |
| Gmail, Docs, Drive, NotebookLM을 같이 쓴다 | Google AI Pro | 저장공간/문서 앱 안 기능을 쓰지 않으면 해지 |
| IDE에서 코드 수정과 리뷰가 많다 | Copilot Pro | 자동완성만 쓰고 agent/code review를 안 쓰면 재검토 |
| 20달러 플랜이 자주 막힌다 | ChatGPT Pro / Claude Max | 막힌 기록이 없으면 보류 |
| 여러 플랜이 다 좋아 보인다 | 하나만 한 달 | 동시에 결제하지 않는다 |

내가 추천하는 방식은 한 달에 하나만 고르는 것이다. 3개를 동시에 결제하면 뭐가 효과가 있었는지 모른다. 첫 주에는 실제로 연 작업을 기록한다. 둘째 주에는 막힌 지점을 적는다. 셋째 주에는 무료 플랜으로 되돌려도 되는지 확인한다. 넷째 주에는 다음 달 유지/교체/해지를 결정한다. AI 구독은 기능표보다 "이번 주에도 내가 열었는가"가 더 정확하다.

## 내 기준

가끔 쓰는 사람은 무료부터다. 매일 한 계정에서 글, 파일, 이미지, 리서치를 섞으면 ChatGPT Plus가 첫 후보가 된다. 긴 문서와 차분한 문장 검토가 많으면 Claude Pro를 한 달 테스트한다. Google 문서, 메일, 드라이브에 이미 돈을 쓰고 있으면 Google AI Pro를 저장공간까지 포함해서 계산한다. 코딩이 IDE와 GitHub 안에서 대부분 끝나면 Copilot Pro부터 본다.

월 `$100` 이상은 "좋아 보인다"가 아니라 "막혀서 손실이 난다"일 때다. ChatGPT Pro, Claude Max, Google Ultra, Copilot Max는 모두 사용량과 고급 기능을 사는 쪽이다. 하루에 몇 시간씩 붙어 있고, 제한 때문에 실제 작업이 끊긴 기록이 있을 때 후보가 된다. 아니면 그 돈은 필요한 앱 하나, 저장공간, 백업 장치, 도메인·호스팅 비용으로 쓰는 편이 더 낫다.

결론은 단순하다. AI 플랜은 제일 똑똑한 모델을 고르는 문제가 아니라 내 반복 작업이 어느 화면에서 끝나는지 고르는 문제다. 브라우저, 문서, IDE, 저장공간, agent workflow 중 하나를 먼저 고르면 가격표가 덜 흔들린다.
