## 추천만 있으면 내 선택이 약해진다

금고 번호 맞추기는 세 자리 번호를 넣고 `몇 자리가 맞았는지`, `숫자는 맞지만 자리가 다른 게 몇 개인지` 보면서 좁혀가는 게임이다. 이전 판에는 추천 후보가 있었다. 남은 경우를 많이 줄일 것 같은 번호를 골라주는 기능이다.

그런데 추천만 보이면 사용자가 직접 고른 번호가 좋은지 나쁜지 알기 어렵다. 결국 “추천 누르면 되나?”가 되어버린다. 추론 게임이면 내가 고른 번호도 판단할 수 있어야 한다.

## 이번에 넣은 것

- 현재 입력한 세 자리 번호가 평균 몇 개 후보를 남길지 계산한다
- 운이 나쁠 때 최악으로 몇 개 후보가 남는지도 같이 보여준다
- 중복 숫자, 이미 본 번호, 기존 기록과 충돌하는 번호는 미리 빨간 문구로 걸러낸다
- 추천 후보는 그대로 두되, 사용자가 직접 고른 번호와 비교할 수 있게 했다
- 시작 문구도 “추천 후보를 눌러라”보다 “얼마나 줄이는지 보고 눌러보기” 쪽으로 바꿨다

큰 규칙은 그대로다. 다만 이제 번호를 찍기 전에 한 번 생각할 거리가 생겼다. 평균 후보 수가 낮고 최악 후보 수도 낮으면 꽤 괜찮은 시도다. 반대로 숫자가 커 보이면 그 번호는 정보를 별로 못 줄일 가능성이 높다.

## 다음에 볼 것

다음에는 힌트 기록을 더 읽기 쉽게 만들고 싶다. 예를 들면 `123 → 1자리 1숫자`를 그냥 줄로 남기는 대신, 어떤 숫자가 후보로 살아남았는지 조금 더 분명하게 표시하는 방식이다. 그래야 금고 게임이 숫자 맞추기보다 작은 추론판처럼 느껴진다.
