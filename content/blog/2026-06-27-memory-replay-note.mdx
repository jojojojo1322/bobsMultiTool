## 외우는 게임에는 숨 돌릴 틈이 필요하다

불빛 패턴 메모리는 단순하다. 칸이 빛나는 순서를 보고 그대로 누르면 된다. 그런데 한 번 놓치면 너무 바로 실패 쪽으로 기울었다. 특히 모바일에서 손가락을 옮기는 사이에 “방금 몇 번이었지”가 되면 다시 볼 방법이 없었다.

틀리면 같은 패턴을 자동으로 다시 보여주긴 했지만, 그건 이미 틀린 뒤의 이야기다. 사용자가 스스로 멈추고 다시 볼 수 있어야 기억 게임 느낌이 더 산다.

## 이번에 넣은 것

- `R` 키로 현재 패턴을 다시 보여준다
- 모바일에서도 누를 수 있게 `다시 보기` 버튼을 추가했다
- 다시 보기는 공짜가 아니라 작은 점수와 집중도를 깎는다
- 방향키/WASD, Space/Enter, 숫자 1~9 입력은 그대로 유지했다
- 게임 안 문구도 “보고 외우고 누르기”보다 “헷갈리면 다시 보기”가 보이게 바꿨다

큰 기능은 아니다. 그래도 이 한 버튼 때문에 플레이가 조금 덜 벌칙처럼 느껴진다. 기억이 애매할 때 바로 누를 수 있는 안전장치가 생겼다.

## 다음에 볼 것

다음에는 입력한 순서가 너무 조용히 지나가지 않게, 마지막으로 누른 칸과 다음 입력 위치를 더 분명하게 보여주는 쪽을 볼 만하다. 지금은 패턴을 외우는 맛은 있는데, 내가 어디까지 맞췄는지 순간적으로 놓칠 때가 있다.
